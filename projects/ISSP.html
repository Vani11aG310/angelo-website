ThingsMyKidsDo

For this project, I was tasked with the development of a prototype application that adapted the ThingsMyKidsDo website, now Happyway.ca, to work with mobile devices. During this project, I lead a team of full-stack engineer students, utilizing the Agile development method to manage our project lifecycle. Front-end development was done in React Native and Express.js for back-end Deployed on AWS EC2 and Heroku. Project was managed using an Agile environment, utilizing sprints, daily stand-ups, client demos, and using Github to manage the codebase.
